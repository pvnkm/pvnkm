### 🚀 Featured Projects

<details>
<summary><b>🧠 DocMind — Production-Grade RAG Chatbot</b></summary>
<br/>

A full-stack document Q&A system where answers are strictly grounded in uploaded documents (PDF, Word, CSV, TXT) via a complete retrieval-augmented generation pipeline with conversational memory.

| Aspect | Detail |
|---|---|
| **Stack** | LangChain · FastAPI · Streamlit · ChromaDB · Groq (LLaMA 3.3 70B) · HuggingFace Embeddings |
| **Scale** | Multi-format document ingestion with per-session isolation |
| **Performance** | MMR retrieval (`k=5`, `fetch_k=10`) for diverse, non-redundant context |
| **Security** | In-memory vector store per session — no disk accumulation, no cross-session leakage |
| **Impact** | Flagship Gen AI portfolio project demonstrating production RAG design decisions |
| **Repository** | [github.com/pavankumarb/docmind](https://github.com/pavankumarb) |

Built with a manual `build_question()` context-prepending approach (replacing `ConversationalRetrievalChain`) to fix stale session bleed-across conversations — a deliberate architectural choice over the default LangChain abstraction, prioritizing correctness over convenience.

</details>
